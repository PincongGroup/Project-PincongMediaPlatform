# 關於我們

品葱月刊是一個由熱心葱友組成的媒體，旨在通過各種報導和分析介紹普世價值、中共暴政對所有人的危害及如何不助紂為虐的方法。

在此我們承諾：
1. 拒絕假新聞
2. 拒絕販賣焦慮恐慌
